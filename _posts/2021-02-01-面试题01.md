## leetcode 面试题01 判定字符是否唯一

实现一个算法，确定一个字符串 s 的所有字符是否全都不同。

示例1:

```bash
输入: s = "leetcode"
输出: false 
```

示例2:

```bash
输入: s = "abc"
输出: true
```

### 思路

使用 Set 数据结构去掉重复项

### 代码

```js
var isUnique = function(astr) {
    return new Set(astr).size === astr.length;
};
```

### 复杂度分析

- 时间复杂度: O(N)
- 空间复杂度: O(N)
