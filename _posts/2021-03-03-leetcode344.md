## leetcode 344题 反转字符串

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 char[] 的形式给出。

不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用 O(1) 的额外空间解决这一问题。

你可以假设数组中的所有字符都是 ASCII 码表中的可打印字符。

示例1:

```bash
输入：["h","e","l","l","o"]
输出：["o","l","l","e","h"]
```

示例2:

```bash
输入：["H","a","n","n","a","h"]
输出：["h","a","n","n","a","H"]
```

### 思路

双指针

### 代码

```js
var reverseString = function(s) {
    let left = 0, right = s.length-1;
    while (left <= right) {
        let temp = s[left];
        s[left] = s[right];
        s[right] = temp;
        left++;
        right--;
    }
    return s;
};
```

### 复杂度分析

- 时间复杂度: O(N))
- 空间复杂度: O(1)
